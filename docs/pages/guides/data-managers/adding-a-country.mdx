# Adding a Country

This page will cover:
- What "adding a country" means: running the hex pipeline to generate H3 hex cells with population and people group data
- Prerequisites: WorldPop raster data availability, ISO country codes, center coordinates
- The pipeline steps: WorldPop download, H3 hex cell generation, people group classification, dashboard registration
- Using the `gc-add-country` skill or running `add_country.sh` directly
- Verifying the result: checking that the engagement dashboard renders the new country correctly
- Post-pipeline steps: committing generated data, deploying the updated dashboard

> Content will be provided by the documentation content authors.
