# Knowledge Graph Overview

This page will cover:
- What's in the graph: ~3.2M triples spanning people groups, organizations, assessments, resources, and geographic/linguistic/religious classifications
- Data sources and their named graphs: Joshua Project, IMB, HIS Registries (ROP, ROL, ROR, ROG), IMB Resources, Jesus Film, FCBH
- The GC Ontology as the unifying schema: how all sources map to shared classes and properties
- How the two integration patterns feed the graph: bridge materialization (batch) and SPARQL federation (runtime)
- Exploring the graph: using the platform's SPARQL interface or the API gateway's `/v0/sparql` endpoint

> Content will be provided by the documentation content authors.
