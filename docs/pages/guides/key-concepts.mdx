# Key Concepts

This page will cover:
- **Knowledge Graph** — what a triplestore is and why RDF/SPARQL matter for missions data
- **GC Ontology** — the shared schema (OWL + SHACL) that defines classes like PeopleGroup, Organization, Assessment, and MissionResource
- **Named Graphs** — how data from different sources (Joshua Project, IMB, HIS Registries) is isolated and governed
- **People Group in Country (PGIC)** — the fundamental unit of analysis, keyed by ROP3 + ROG3 codes from HIS Registries
- **SKOS Vocabularies** — standardized concept schemes for religions, languages, geography, and assessment methodologies
- **Bridges** — batch data pipelines that transform external source data into RDF and load it into GraphDB
