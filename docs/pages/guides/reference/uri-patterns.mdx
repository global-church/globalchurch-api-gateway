# URI Patterns

This page will cover:
- Ontology namespace: `https://ontology.global.church/core#` (prefix `gc:`), plus `jp:` and `imb:` for source-specific concepts
- Data instance URIs: `https://data.global.church/{source}/{type}/{id}` pattern
- People group URIs: keyed by ROP3 code (e.g., `data.global.church/rop/{ROP3}`)
- Assessment and result URIs: `data.global.church/jp/assessment/{ROP3}-{ROG3}`, `data.global.church/imb/result/{ROP3}-{ROG}`
- HIS Registry URIs: ROL (`data.global.church/his/rol/{iso639-3}`), ROR (`data.global.church/his/ror/{code}`), ROG (`data.global.church/his/rog/{code}`)
- Organization URIs: `data.global.church/org/{slug}`
