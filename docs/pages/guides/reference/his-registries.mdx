# HIS Registries

This page will cover:
- What HIS Registries are: shared code systems maintained by hisregistries.org for interoperability across mission organizations
- **ROP (Registry of Peoples)** — ROP3 is the 6-digit perpetual people group code used as the primary key across the missions data ecosystem
- **ROL (Registry of Languages)** — based on ISO 639-3; 7,083 language concepts in the knowledge graph
- **ROR (Registry of Religions)** — 46 religion concepts in a 2-level SKOS hierarchy (e.g., `ror:M` for Islam, `ror:CRC` for Roman Catholic)
- **ROG (Registry of Geography)** — 269 concepts: 5 regions, 22 subregions, 242 countries in a 3-level hierarchy
- How these registries are represented as SKOS ConceptSchemes in the `his-registries` named graph (~172K triples)
