# MCP Integration

This page will cover:
- What the MCP endpoint is: `POST /v0/mcp` exposes Global.Church data as tools for AI agents
- Available MCP tools: `churches_search_v1` for searching the organization directory
- Authentication: Bearer token or `?apiKey=` query parameter
- Connecting from Claude Desktop, Cursor, or other MCP-compatible AI clients
- Example: configuring an MCP client to search for churches by location, denomination, or name
- Future roadmap: additional MCP tools for SPARQL queries, people group lookups, and resource discovery
