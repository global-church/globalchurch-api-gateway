# SPARQL Quickstart

This page will cover:
- Sending your first SPARQL query via `POST /v0/sparql` with your API key
- The GC namespace prefixes you need: `gc:`, `jp:`, `imb:`, `skos:`, `prov:`
- Example queries: listing people groups by country, finding unreached peoples, querying assessment results
- Understanding the 3-block Assessment pattern: PeopleGroup + Assessment (provenance) + AssessmentResult (metrics)
- Working with named graphs: scoping queries to specific data sources like Joshua Project or IMB
- Accept headers for different response formats: JSON, CSV, Turtle

> Content will be provided by the documentation content authors.
